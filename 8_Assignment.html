HTML (HyperText Markup Language)
The standard markup language for creating web pages and web applications. It defines the structure and layout of content on the web.
CSS (Cascading Style Sheets)
Used to style and format the visual presentation of HTML documents. It enhances the appearance and layout of web pages.
JavaScript
A programming language that enables dynamic, interactive behavior on web pages. It is essential for client-side scripting and building responsive user interfaces.
Node.js
An open-source, server-side JavaScript runtime that allows developers to build scalable and high-performance network applications. It is commonly used for building server-side applications.
Express.js
A web application framework for Node.js, designed for building robust and scalable web applications and APIs. It simplifies the process of creating server-side applications.
MongoDB
A NoSQL database that stores data in JSON-like documents with dynamic schemas. It is commonly used for handling large amounts of unstructured data in web applications.
React.js
A JavaScript library for building user interfaces, particularly for single-page applications where UI updates are frequent. It allows developers to create reusable UI components.
Angular
A TypeScript-based open-source framework for building dynamic web applications. It simplifies the development process and provides a structured approach to building scalable applications.
