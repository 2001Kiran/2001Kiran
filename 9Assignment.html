1. < h1 > to < h6 > ;
    • Used for defining headings with varying levels of importance.
    • Best practice is to use them in a hierarchical order (e.g., <h1> for the main heading, <h2> for subheadings, and so on).
  2. < p >
    • Defines paragraphs of text.
    • Best practice is to use <p> for each block of text to ensure proper structure and styling.
  3. < em > and < strong >
    • <em> is used to emphasize text, typically displayed in italics.
    • <strong> is used to highlight strong importance, typically displayed in bold.
    • Best practice is to use these tags semantically, choosing <em> for emphasis and <strong> for stronger importance.
  4. < spaan >
    • Provides a way to group and apply styles to inline elements without affecting the document's structure.
    • Best practice is to use <span> when applying styles or scripting to specific parts of text within a larger block.
  5. <a>
    • Creates hyperlinks to navigate to other pages or resources.
    • Best practice is to use the <a> tag with the href attribute to define the destination URL.
